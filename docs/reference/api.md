# API Reference

dreamcli exposes three subpath exports, each with a focused API surface.

## Package Exports

| Import             | Purpose                                                          |
| ------------------ | ---------------------------------------------------------------- |
| `dreamcli`         | Schema builders, CLI runner, output, parsing, resolution, errors |
| `dreamcli/testkit` | Test harness, capture utilities, test prompter                   |
| `dreamcli/runtime` | Runtime adapter factory, platform detection                      |
